# ✅ Task Manager (To-Do List 2.0)

Gerenciador de tarefas com estados de conclusão, exclusão e filtros dinâmicos.

### 🛠 Tecnologias e Conceitos:
- **Gerenciamento de Estado:** Lógica para alternar tarefas entre concluídas e pendentes.
- **Filtros Dinâmicos:** Manipulação de arrays para exibir listas personalizadas.
- **Persistência:** Uso de LocalStorage para salvar as tarefas.
